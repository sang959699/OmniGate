iOS Shortcut Setup Guide:

  1. Open the Shortcuts app on iOS and create a new Shortcut.
  2. Search for and add the Get Contents of URL action.
  3. Copy one of the green URLs above and paste it into the URL field. (The dashboard automatically replaces localhost with your server's current IP address).
  4. Expand the action detail and set the Method to POST.
  5. Save and trigger the shortcut by saying "Hey Siri, [Shortcut Name]"!
